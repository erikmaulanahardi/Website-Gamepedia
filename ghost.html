    GHOSTS

    Hantu adalah fokus utama Phasmophobia. Setiap hantu akan memiliki serangkaian karakteristik unik, termasuk pengenal pribadi, kemampuan, dan bukti. Pemain harus menyelidiki beberapa aspek ini untuk mengidentifikasi hantu dengan benar. Hantu tersebut dapat berinteraksi dengan lingkungannya atau menyebabkan kejadian hantu, dan bahkan mencoba membunuh pemain dengan memulai perburuan..

    Banshee

    Evidence :

  • D.O.T.S Projector, Ghost Orb, dan Ultraviolet.

    • Kekuatan :

  • Hanya menargetkan satu player ketika hunt.
  • Meningkatkan peluang munculnya singing ghost event.

    • Kelemahan :

  • Memiliki ratapan khas ketika menggunakan Parabolic Microphone.

    • Demon

    Evidence :

  • Ghost Writing, Ultraviolet, Freezing Temperature.

    • Kekuatan :

  • Dapat memulai hunt lebih sering.
  • Dapat memulai hunt meskipun sanity player masih penuh.

    • Kelemahan :

  • Radius penggunaan Crucifix 50% lebih besar dari hantu lain.

    • Deogen

    Evidence :

  • D.O.T.S Projector, Ghost Writing, Spirit Box.

    • Kekuatan :

  • Selalu mengetahui lokasi player ketika hunt.
  • Bergerak sangat cepat ketika menuju ke tempat player.

    • Kelemahan :

  • Bergerak sangat lambat ketika berada di dekat player.

    • Goryo

    Evidence :

  • D.O.T.S Projector, EMF Level 5, Ultraviolet.

    • Kekuatan :

  • Hanya dapat memasuki D.O.T.S ketika tidak ada orang di sekitarnya.
  • Siluet D.O.T.S hanya dapat dilihat melalui Video Camera.

    • Kelemahan :

  • Tidak bisa berkeliaran jauh dari ruangannya, atau berpindah ruangan favorit.

    • Hantu

    Evidence :

  • Ghost Orb, Ultraviolet, Freezing Temperature.

    • Kekuatan :

  • Semakin rendah suhu ruangannya, semakin cepat pergerakannya.

    • Kelemahan :

  • Daerah yang lebih hangat memperlambat pergerakan Hantu.
  • Player akan menghasilkan nafas dingin saat hunt jika kotak sekring dimatikan.
  • Tidak akan pernah menyalakan kotak sekring.

    • Jinn

    Evidence :

  • EMF Level 5, Ultraviolet, Freezing Temperature.

    • Kekuatan :

  • Bergerak dengan sangat cepat ketika melihat player jika kotak sekring menyala.

    • Kelemahan :

  • Tidak dapat menggunakan kemampuannya jika kotak sekring dalam keadaan mati.
  • Tidak akan pernah mematikan kotak sekring secara langsung.

    • Mare

    Evidence :

  • Ghost Writing, Ghost Orb, Spirit Box.

    • Kekuatan :

  • Berpeluang besar untuk memulai hunt di tempat gelap.
  • Kadang-kadang akan langsung mematikan lampu.
  • Tidak akan pernah menyalakan lampu.

    • Kelemahan :

  • Menyalakan lampu akan mengurangi kemungkinan terjadinya hunt.

    • Moroi

  • Ghost Writing, Freezing Temperature, Spirit Box.

    • Kekuatan :

  • Bergerak lebih cepat ketika sanity player rendah.
  • Dapat mengutuk player, membuat mereka kehilangan sanity lebih cepat dari biasanya saat berada di zona investigasi.

    • Kelemahan :

  • Incense membutakan hantu 50% lebih lama selama hunt.

    • Myling

    Evidence :

  • Ghost Writing, EMF Level 5, Ultraviolet.

    • Kekuatan :

  • Menghasilkan suara yang lebih pelan saat hunt.

    • Kelemahan :

  • Menghasilkan suara paranormal lebih sering pada Parabolic Microphone.

    • Obake

    Evidence :

  • Ghost Orb, EMF Level 5, Ultraviolet.

    • Kekuatan :

  • Memiliki peluang untuk tidak meninggalkan fingerprint saat berinteraksi dengan sesuatu.
  • Fingerprint terkadang lebih cepat hilang.

    • Kelemahan :

  • Memiliki kemungkinan kecil untuk meninggalkan fingerprint khusus.
  • Dapat dengan cepat berubah bentuk menjadi model lain saat hunt.

    • Phantom

  • D.O.T.S Projector, Spirit Box, Ultraviolet.

    • Kekuatan :

  • Melihat Phantom akan menurunkan sanity player secara signifikan.

    Evidence :

    • Kekuatan :

    • Kelemahan :

  • Menjadi tidak berdaya jika tidak ada objek yang dapat dilemparkan di sekitarnya.

    • The Twins

    Evidence :

  • Freezing Temperature, EMF Level 5, Spirit Box.

    • Kekuatan :

  • Sering berinteraksi dengan lingkungan pada saat yang sama, tetapi biasanya di tempat yang berbeda.

  • Dapat berteleportasi ke player acak.

    • Kelemahan :

  • Wraith takut pada salt dan akan secara aktif mengindarinya.

    • Yokai

    • Kekuatan :

    • Kelemahan :

  • Saat berburu, yokai hanya bisa mendengar suara yang dekat dengannya.

    • Yurei

    • Kekuatan :

  • Memiliki efek yang lebih kuat pada sanity manusia.

  • Menggunakan incense di dalam ruangan yang sama dengan Yurei akan menjebaknya untuk sementara, Menggunakan kemampuannya dapat menyebabkannya menutup semua pintu.
